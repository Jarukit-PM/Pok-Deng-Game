# Pok Deng Game

A simple Pok Deng card game. React frontend + Express API.

## Run

**1. Start the API**

```bash
cd api
npm install
npm run dev
```

Runs on `http://localhost:8080`

**2. Start the app**

```bash
cd app
npm install
npm run dev
```

Open the URL shown in the terminal (usually `http://localhost:5173`).

## Play

1. Enter starting chips and click **Start Game**
2. Cut the deck, place a bet, then **Draw** or **Stay**
3. Beat the dealer to win chips
